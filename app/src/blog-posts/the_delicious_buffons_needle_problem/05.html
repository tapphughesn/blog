Estimating Pi

Pi day, which was just a few days ago, is often celebrated by calculating digits of pi--an activity that we all love to do4. What if I told you that we can use these simulations to do it? The probability that a needle crosses one of the lines, p, can be approximated by counting the number of crossings and dividing by the total number of needles:

p = \frac{2l}{d \pi} \approx \frac{\text{\# crossings}}{\text{\# needles}}

Rearranging that equation, we can get an estimate for pi:

pi \approx \frac{2l(\text{\# needles})}{d(\text{\# crossings})}

If you set l = ½ d, this simplifies to

pi \approx \frac{\text{\# needles}}{\text{\# crossings}}
